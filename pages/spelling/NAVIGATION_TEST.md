## 技术实现要点

### 导航栏高度计算
```css
padding-top: calc(var(--status-bar-height, 44px) + 88rpx + 40rpx);
```
- `--status-bar-height`: 系统状态栏高度（动态获取）
- `88rpx`: 导航栏内容区域高度
- `40rpx`: 额外的内容间距

### 按钮交互设计
- 圆形按钮 (`border-radius: 50%`)
- 半透明背景 (`rgba(0, 0, 0, 0.05)`)
- 按压缩放效果 (`transform: scale(0.95)`)

### 布局稳定性
- 使用 `z-index: 1000` 确保导航栏层级最高
- 绝对定位的左右按钮不会影响标题居中
- 固定定位避免页面滚动时导航栏移动
